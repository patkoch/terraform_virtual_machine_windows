# Terraform Commands

## Initialization

Use the Init command for establishing a connection to the backend, use:

```
terraform init
```

## Validation

Use the Validation command for verifying the file

```
terraform validation
```

## Apply

Use the Apply command for conducting the provisioning

```
terraform apply
```
